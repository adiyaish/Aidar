cd-перемешает нас в какую то папку директоию 
ls просматривает содержимое диретории
touch name создает файл
mkdr создает папку 
ls -a- показывает файлы вместе со скрытыми файлами и папками
ls -l-показывает файлы и папки с доп информацией 
ls-la-показывает файлы и папки со скрытыми файлами и папками +доп информация
cd-переходит в домашнюю папку 
cd~- переходит в домашнюю папку
cd .-переходит в ту папку в которой вы находитесб
cd ..-переходит в предыдущую папку ( родительска папка)
rm name- удаление файла name
rmdir-удаление пустой диреkтории (папки)
rm -rf- удаляет все; папки содерживым, фвйлы, пустыу папки
history-выводит историю команд
history  какое то число- выводит историю в каком тот количестве
history |grep слово- выводит все команды, в которых есть указанное слово
pwd-показывает абсолютный путь (начиная с корневой папки) путь до текущей директории в которой вы находитесь
cat- выводит содержимое файла
nano-текстовый редактор, при помощи которого мы можем записать данные в файл
mv test1.txt test3.txt - переименование файл test1.txt в test3txt
mv test1 test3-переименовывает папку  test1 B test3
mv test1.txt test1-перемещает файл test1.txt в папку  test1
apt-системный установщик пакетов
snap- установщик пакетов ( чаще всего используют для установки программ)
 sudo (superuser do)-выполнение какой-либо команды от имени администратора
